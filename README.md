## Setup Instructions

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   ```

2. **Navigate to the code directory**
   ```bash
   cd cousing/code
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```
